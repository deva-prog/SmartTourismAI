# 🌍 SmartTourismAI

## AI-Powered Tourism & Heritage Platform

SmartTourismAI is a tourism and heritage technology platform that helps users discover tourist destinations, explore cultural information, generate travel itineraries, and get personalized travel recommendations.

## 🚀 Features

* Travel Recommendation System
* Cultural Information Cards
* Dynamic Trip Itinerary Generator
* Tourism Dashboard
* Interactive Chatbot
* User-Friendly Streamlit Interface

## 🛠️ Technology Stack

* Python
* Streamlit
* Pandas
* Matplotlib

## 📊 Project Modules

* Tourist Places Dataset
* Recommendation Engine
* Cultural Information Cards
* Dashboard Visualization
* AI Travel Planner
* Tourism Chatbot

## 🎯 Problem Statement

Travelers often face difficulty finding tourist destinations, cultural heritage information, and travel planning tools on a single platform.

## 💡 Solution

SmartTourismAI combines destination recommendations, cultural information, dashboards, chatbot support, and itinerary generation into one easy-to-use platform.

## ▶️ Run Locally

pip install -r requirements.txt

streamlit run app.py
